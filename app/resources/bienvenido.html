Ahora puede comenzar a crear sus clases en la estructura de la plantilla.

        
  1.      Crea la clase de controlador. Ejemplo: ProductForm, ProductList;     
    2.     
  2.      Inicie sesión como administrador (admin), para registrar los permisos;     
    3.     
  3.      Registre la clase como un programa (Administración -> Programas). Debe completar el campo "controlador" con el nombre de clase (Ej: ProductForm);     
    4.     
  4.      Dar permiso en la clase registrada a algún usuario o grupo (Administración -> Usuarios o Administración -> Grupos);     
    5.     
  5.      Agregue el programa al menu.xml, que contiene la estructura del menú;     
    6.     
  6.      Realice un cierre de sesión e inicie sesión con algún usuario con permiso para ejecutar el programa creado;     
    7.     
  7.      Termina la creación del programa.
